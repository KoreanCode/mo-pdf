# PDF 영역 블러 앱 사용 설명서

이 앱은 PDF 또는 이미지 파일을 업로드한 뒤 원하는 영역만 직접 지정해 블러 처리하는 로컬 프로그램입니다. 파일은 내 컴퓨터에서 처리되며, 기본 주소는 `http://127.0.0.1:5000`입니다.

## 1. 압축 해제 후 폴더 구조

```text
pdf-region-blur-app
├─ Windows
│  ├─ start_windows.vbs
│  ├─ launcher_windows.bat
│  └─ 앱 실행 파일들
├─ macOS
│  ├─ start_mac.command
│  ├─ launcher_mac.command
│  ├─ build_mac_dmg.command
│  └─ 앱 실행 파일들
├─ README.txt
├─ USER_GUIDE_KO.txt
└─ docs
```

본인이 사용하는 운영체제 폴더만 열면 됩니다.

## 2. 런처에서 실행하기

런처에는 세 개의 버튼이 있습니다.

- `서버 실행`: 로컬 서버를 시작합니다. 처음 실행하면 필요한 패키지를 자동으로 설치합니다.
- `접속하기`: 브라우저에서 `http://127.0.0.1:5000`을 엽니다.
- `서버 종료`: 실행 중인 로컬 서버를 종료합니다.

실행 로그는 각 운영체제 폴더 안의 `logs` 폴더에 저장됩니다.

## 3. Windows에서 실행하기

1. `Windows` 폴더를 엽니다.
2. `start_windows.vbs`를 더블클릭합니다.
3. `O'range PDF Blur` 런처가 열리면 `서버 실행`을 누릅니다.
4. 서버 실행 상태가 표시되면 `접속하기`를 누릅니다.
5. 사용이 끝나면 런처에서 `서버 종료`를 누릅니다.

Python 3가 없으면 https://www.python.org/downloads/ 에서 설치한 뒤 다시 실행하세요. 설치할 때 `Add python.exe to PATH` 옵션을 선택하면 문제를 줄일 수 있습니다.

## 4. macOS에서 실행하기

1. `macOS` 폴더를 엽니다.
2. `start_mac.command`를 더블클릭합니다.
3. `O'range PDF Blur` 런처가 열리면 `서버 실행`을 누릅니다.
4. 서버 실행 상태가 표시되면 `접속하기`를 누릅니다.
5. 사용이 끝나면 런처에서 `서버 종료`를 누릅니다.

macOS에서 권한 안내가 나오면 다음 중 하나를 사용하세요.

### 방법 A: 오른쪽 클릭으로 열기

1. `start_mac.command`를 오른쪽 클릭합니다.
2. `열기`를 선택합니다.
3. 다시 `열기`를 선택합니다.

### 방법 B: 실행 권한 부여

터미널을 열고 `macOS` 폴더로 이동한 뒤 아래 명령을 한 번 실행합니다.

```bash
chmod +x start_mac.command launcher_mac.command build_mac_dmg.command run_mac.command stop_mac.command
```

## 5. macOS DMG 만들기

DMG 파일은 Mac에서 만들어야 합니다.

1. Mac에서 ZIP 압축을 풉니다.
2. `macOS/build_mac_dmg.command`를 실행합니다.
3. 상위 폴더에 `O-range-PDF-Blur-macOS.dmg`가 생성됩니다.

DMG 안의 `.app`을 실행하면 런처가 열립니다. 앱은 실행 파일을 사용자 홈의 `~/Library/Application Support/OrangePdfBlur`로 복사해 사용하므로 DMG 안에서 바로 실행해도 서버 로그와 가상환경을 쓸 수 있습니다.

## 6. PDF 블러 처리 방법

1. 웹 화면에서 `편집 시작` 버튼으로 PDF 또는 이미지를 업로드합니다.
2. 미리보기에서 블러 처리할 영역을 마우스로 드래그합니다.
3. 여러 영역을 지정할 수 있습니다.
4. 잘못 지정한 경우 `마지막 영역 삭제` 또는 `전체 삭제`를 누릅니다.
5. 필요하면 `블러 강도` 값을 조정합니다.
6. `선택 영역 블러 PDF 다운로드`를 누릅니다.

다운로드된 PDF는 이미지 기반 PDF입니다. 원본의 선택 가능한 텍스트, 링크, 주석, 양식 필드는 유지되지 않습니다.

## 7. 키보드로 영역 지정하기

- `Enter` 또는 `Space`: 새 블러 영역 추가
- 방향키: 마지막으로 추가한 영역 이동
- `Shift` + 방향키: 마지막으로 추가한 영역 크기 조정
- `Delete` 또는 `Backspace`: 마지막으로 추가한 영역 삭제

## 8. 자주 발생하는 문제

### 런처가 열리지 않음

Python 3가 설치되어 있는지 확인하세요. Windows에서는 Python 설치 시 PATH 옵션을 켜는 것이 좋습니다.

### 서버 실행 중 패키지 설치 실패

인터넷 연결을 확인한 뒤 다시 실행하세요. 회사 또는 학교 네트워크에서는 보안 정책 때문에 설치가 막힐 수 있습니다.

### 브라우저가 열리지 않음

브라우저 주소창에 아래 주소를 직접 입력하세요.

```text
http://127.0.0.1:5000
```

### 서버 종료가 되지 않음

런처의 `서버 종료`를 다시 누릅니다. 그래도 종료되지 않으면 Windows에서는 `stop_windows.vbs`, macOS에서는 `stop_mac.command`를 실행하세요.

## 9. 제한 사항

- 암호화된 PDF는 처리하지 않습니다.
- 매우 큰 PDF는 미리보기 생성과 출력 처리에 시간이 걸릴 수 있습니다.
- 출력 PDF는 이미지 기반 PDF입니다.
- 원본 텍스트 레이어, 링크, 주석, 양식 필드는 보존되지 않습니다.
